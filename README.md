Веб сайт с вопросами, который можно скачать как вордовский документ
Для запуска:
  Открыть терминал
  Перейти в папку проекта 
  Ввести в терминал команду "python app.py"
  Перейти по ссылке (откроется браузер с формой)
После ввода ответов, по нажатию кнопки происходит скачивание Ворд файла с введенными данными
