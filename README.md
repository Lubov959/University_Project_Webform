# Веб‑приложение для создания анкет в формате Word

Веб‑приложение, позволяющее пользователям отвечать на вопросы в форме, а затем скачивать заполненные данные в виде документа Microsoft Word (.docx).

## Авторы

* Веригина Любовь
* Кудряшова Александра

## Функциональность

Приложение предоставляет:

1. Веб‑форму с набором вопросов.
2. Возможность выбирать цвет из палитры для оформления.
3. Функцию генерации и скачивания документа Word с введёнными ответами и выбранным цветом.

## Установка и запуск

### Предварительные требования

* Установленный **Python** (версия 3.13 или выше).
* Установленный менеджер пакетов **pip**.

### Пошаговая инструкция

1. **Клонируйте репозиторий** или скачайте архив с проектом на свой компьютер.
2. **Откройте терминал** (командную строку) и перейдите в папку с проектом:
   
  **_bash_**
  
  		cd путь/к/папке/проекта
   
3. **Установите все необходимые зависимости из файла requirements.txt:**
   
  **_bash_**
   
		pip install -r requirements.txt

4. **Запустите приложение:**
   
**_bash_**

		python app.py

5. После запуска в терминале появится сообщение со ссылкой (обычно _http://127.0.0.1:5000_ или _http://localhost:5000_). **Откройте эту ссылку в браузере.**

### Использование
1. На открывшейся веб‑странице вы увидите форму с вопросами и палитрой цветов.
2. Заполните все поля формы, вписав ответы на вопросы в соответствующие поля.
3. Выберите нужный цвет, нажав на один из доступных вариантов в палитре.
4. После заполнения всех данных нажмите кнопку «Сгенерировать документ» (или аналогичную).
5. Браузер автоматически начнёт скачивание файла, содержащего ваши ответы и информацию о выбранном цвете.
